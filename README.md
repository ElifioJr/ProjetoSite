# ProjetoSite
## Tecnologia Utilizada
.NET 8.0 utilizando a linguagem C#

## Descrição do projeto
Projeto com base em estudo da liguagem , vinculação com banco de dados SQL utilizando Entity. 
O banco de dados é gerado como localhost, o front-end é o proprio template gerado pelo VisualStudio.

## Técnologias Utilizadas no projeto:
.NET 8.0
Entity Framework
SQLServer
SQLServer Management Studio
BootStrap
